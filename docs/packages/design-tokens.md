# Design Tokens Package

The shared token layer defines:

- color roles
- type scale
- spacing scale
- radii
- shadows
- motion timings
- surface behavior

## Why tokens instead of shared UI components

Tokens travel cleanly across Astro, Nuxt, Next.js, SvelteKit, and VitePress. Component trees do not.

The starter family keeps the visual language shared while leaving rendering native to each framework.
